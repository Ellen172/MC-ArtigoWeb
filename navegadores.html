Suporte dos Navegadores para Html e Css

Antes de começar, é importante saber que o o suporte das linguagens não é feito pelo navegador em si, mas pelo motor de renderização que ele possui. Dessa forma os navegadores Chrome e Safari terão a mesma renderização já que ambos possuem o motor Webkit, enquanto que para os demais navegadores os motores são: Gecko para o Firefox, Trident para o Internet Explorer e Presto para o Opera.

Atualmente o motor Webkit é o que melhor renderiza os códigos em html e css.

Existem alguns sites que são muito bons para demostrar o nivel de suporte do navegadores, como por exemplo o Html5 & Css3 Readiness, que contém um infográfico com cada navegador e cada recurso, há também o Html5 Test que dá uma pontuação para o navegador que você estiver usando quando o abrir, o CanIUse que possui informações mais completas sobre o suporte das linguagens.
