Principais tags html

Tags Estruturais

Html: envolve todo o documento html

Head: envolve o cabeçalho do dodumento

Meta: fornece informações gerais sobre o documento

Style: informações de estilo

Title: o título do documento

Body: envolve o corpo (texto e tags) do documento html

Paragrafo

p: um paragrafo simples

span: um paragrafo para se colocar um estilo mais marcante

Links

a: cria um link e inclui atributos em comum

href: o url do documento que será vinculado a este a

Listas

ol: define uma lista ordenada

ul: define uma lista não ordenada

li: define um elemento de lista

dl: define uma lista de definções

dt: define o titulo de uma lista de definições

dd: define o texto de definição da lista de definições
